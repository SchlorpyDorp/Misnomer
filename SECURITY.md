# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Misnomer, please do **not** publish it immediately.

Instead, open a GitHub Discussion or contact me through GitHub so I can investigate and fix the issue before it is publicly disclosed.

Please include:

- A description of the issue
- Steps to reproduce it
- The affected version(s), if known
- Any relevant logs or screenshots

Thank you for helping improve Misnomer.
